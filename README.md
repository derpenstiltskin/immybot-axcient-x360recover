# Axcient x360Recover Integration for ImmyBot

> [!WARNING]  
> This integration is still considered Alpha status even though it is feature complete, due to the age and amount of testing completed.

# Features

- Importing clients from the Axcient x360Recover API
- Importing devices from the Axcient x360Recover API
- Automatching of ImmyBot computers and Axcient x360Recover devices
- Automatically grabbing the client install token from the Axcient x360Recover API for agent installs
- Supports installing D2C and Appliance agents
- Supports uninstalling agents
- Supports enabling/disabling agent tray icon
- Supports the Axcient x360Recover 2.x agent (No ARM support), as that's the agent Axcient still recommends and makes publicly available
- HTTP 429 Response handling with exponential backoff
